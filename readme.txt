ToolPress lets you manage third-party tools into your WordPress site.

== Description ==
ToolPress lets you manage third-party tools into your WordPress site. Enable or disable integrations like Google Tag Manager, HubSpot, jQuery, Bootstrap, Font Awesome, Tawk.to chat, and more from a simple admin dashboard.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/toolpress` directory, or install via the WordPress plugin repository.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to 'ToolPress' in the admin menu to configure settings and add code snippets.

== Frequently Asked Questions ==

= Is ToolPress safe for non-developers? =
Yes, the UI is designed for both developers and site owners. All code is sanitized and validated.
